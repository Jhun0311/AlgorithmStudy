Week02 TIL

학습 주제

-   Stack
-   Queue
-   정렬 알고리즘

학습 내용

이번 주에는 자료구조인 Stack과 Queue의 개념 및 동작 원리를 학습하였다. Stack의 후입선출(LIFO) 구조와 Queue의
선입선출(FIFO) 구조를 이해하고 Python을 활용하여 직접 구현해보았다. 또한 선택 정렬과 버블 정렬의 동작 과정을 학습하며
정렬 알고리즘의 기본 원리를 익혔다. 이후 자료구조와 정렬 알고리즘을 활용한 문제를 풀이하며 알고리즘 구현 능력을
향상시켰다.

실습 코드

-   w2.py

어려웠던 점

정렬 알고리즘의 반복문 구조와 데이터가 교환되는 과정을 처음에는 이해하기 어려웠다. 또한 Stack과 Queue가 비슷해 보였지만
실제 동작 방식과 사용 목적이 다르다는 점을 구분하는 데 시간이 필요했다.

해결 방법

교재의 예제 코드를 직접 실행해보며 자료의 이동 과정을 단계별로 확인하였다. 또한 팀원들과 풀이 과정을 공유하고 다양한
예제를 반복적으로 구현하면서 Stack, Queue, 정렬 알고리즘의 동작 원리를 익힐 수 있었다.

느낀 점

자료구조와 정렬 알고리즘은 다양한 알고리즘 문제를 해결하기 위한 기초 개념이라는 것을 알게 되었다. 단순히 코드를 암기하는
것이 아니라 각 자료구조와 알고리즘이 어떤 상황에서 사용되는지 이해하는 것이 중요하다고 느꼈다. 앞으로도 꾸준한 문제
풀이를 통해 알고리즘 활용 능력을 향상시키고 코딩테스트에 대비하고자 한다.
